CodeFlight

CodeFlight is an iOS programming language trainer. Our app uses Highlightr for the code editor, IQKeyboardManager for a scrollable code editor, and GCDWebServer for a live HTML web server.

We consider CodeFlight as RetriceCode's successor as we provide what they once wanted to provide to others.

Usage

CodeFlight provides two classes inside this update:

HTML

This class helps you learn HTML in a simple matter. We provide 12 HTML lessons currently.

You can choose whichever lesson you want to learn from with a simple tap on your screen.

CSS

This class helps you learn CSS in a simple matter. We provide 5 CSS lessons currently.

You can choose whichever lesson you want to learn from with a simple tap on your screen.

iPad Support

CodeFlight also provides iPad support to those who don't own an iPhone or prefer iPads.
